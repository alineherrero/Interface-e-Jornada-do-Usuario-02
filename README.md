# Interface-e-Jornada-do-Usuario-02
Projeto de UX prototipado no Figma: portal desktop de autoatendimento estudantil para abrir, acompanhar e gerenciar solicitações acadêmicas. O objetivo é reduzir a dependência de suporte humano, simplificar processos administrativos e oferecer autonomia com fluxos guiados, feedback claro e navegação intuitiva.
